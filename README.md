# React Clock Demo

一个基于 React 的简单 Web 应用，展示动态时钟和 SVG 图形渲染。

## 功能特性

- **Hello World 页面**：简洁的欢迎界面
- **动态数字时钟**：实时显示当前时间
- **SVG 模拟时钟**：使用 SVG 绘制的动态模拟时钟，包含时针、分针和秒针
- **响应式设计**：适配不同屏幕尺寸

## 技术栈

- **前端**：React 18
- **样式**：CSS3 with gradient backgrounds

## 安装和运行

### 1. 安装依赖

在项目根目录运行以下命令来安装所有客户端依赖：

```bash
npm install
```

### 2. 启动开发服务器

安装完成后，运行以下命令来启动开发服务器：

```bash
npm start
```

这将启动 React 开发服务器，通常在 `http://localhost:3000` 上可用。
